# partypad-x7q2k9 🎉

A private birthday party website, built (vibecoded) by an 11-year-old.

- **Live at:** https://thehaps.app/partypad-x7q2k9/
- **Hidden:** has a `noindex` tag so it stays out of Google. Share the link only with family.
- **How it works:** edit `index.html`, commit, and push. The live site updates in ~30 seconds.
- Pure static HTML/CSS hosted on GitHub Pages — no passwords, no database, nothing to break.
